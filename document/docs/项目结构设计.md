# 项目结构设计

## 完整项目结构

```bash
src/
├── api/                  # API 接口层
├── assets/               # 资源文件
│   ├── styles/           # 样式文件
│   └── images/           # 图片资源
├── components/           # 通用组件
│   ├── common/           # 公共组件
│   └── business/         # 业务组件
├── composables/          # 组合式函数
├── constants/            # 常量定义
├── hooks/                # 自定义钩子
├── layouts/              # 布局组件
├── router/               # 路由配置
├── stores/               # 状态管理
├── types/                # TypeScript 类型定义
├── utils/                # 工具函数
└── views/                # 页面视图
```
